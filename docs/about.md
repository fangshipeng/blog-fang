# 关于本站

这是一个专注于前端技术学习的博客站点。

## 📚 内容方向

- **前端基础**：HTML、CSS、JavaScript 核心知识
- **TypeScript**：类型系统、高级类型、最佳实践
- **工程化**：构建工具、代码规范、测试、CI/CD
- **前沿技术**：Web Components、Server Components、AI 与前端

## 🎯 目标

每天更新一篇高质量技术文章，帮助开发者：

1. 夯实前端基础
2. 掌握现代工程化实践
3. 了解行业前沿动态

## 🛠️ 技术栈

- [VitePress](https://vitepress.dev/) - 基于 Vite 的静态站点生成器
- [Vue 3](https://vuejs.org/) - 渐进式 JavaScript 框架
- [GitHub Pages](https://pages.github.com/) - 静态站点托管
